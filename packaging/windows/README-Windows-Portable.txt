StoryForge 故事熔炉 · Windows 免安装版

使用方法：

1. 解压整个 ZIP。
2. 双击 StoryForge.exe。
3. 浏览器会自动打开：
   http://127.0.0.1:1111/storyforge/
4. 使用期间不要关闭 StoryForge.exe 的黑色窗口；关闭窗口即停止本地服务。

重要说明：

- 这个版本不需要安装 Node.js，也不需要运行 npm install。
- 作品数据默认保存在当前浏览器的 IndexedDB 中。
- 更新新版时，通常只需要下载新版 ZIP 并替换程序文件；作品数据仍在浏览器里。
- 为了安全，更新前建议在「数据管理」里导出 JSON 备份。
- 不要清理浏览器网站数据；清理后本地作品可能会消失。
- 尽量固定使用同一个浏览器和同一个地址：
  http://127.0.0.1:1111/storyforge/

如果无法打开：

- 确认没有安全软件拦截 StoryForge.exe。
- 如果提示 1111 端口被占用，先关闭其他 StoryForge 窗口或占用端口的软件，再重新打开。
- 如果浏览器没有自动弹出，可以手动复制上面的地址到浏览器。
