# StoryForge 故事熔炉 · 如何打开项目

> 给完全没接触过代码的朋友。三种方式，从最省事到最完整，照着做就行。

---

## 🌐 方式一：直接用网页版（最省事，不用下载任何东西）

浏览器打开这个网址，**打开就能用**：

### 👉 https://yuanbw.vercel.app/storyforge/

不用安装、不用敲命令。功能和下载到本地的版本**完全一样**。

> 数据存在你自己的浏览器里（不上传服务器）。换电脑/换浏览器不会自动同步，重要内容请用工具里的「导出」存成文件备份。
> **大多数只想写小说的朋友，用这个就够了。**

---

## ⚡ 方式二：一键启动（想在自己电脑上跑，最简单的本地方法）

我们准备了一个"一键启动"文件，**双击它**，它会自动检查并帮你装好运行环境、然后打开浏览器。你基本不用敲任何命令。

### 准备：先下载项目

1. 浏览器打开 👉 https://github.com/yuanbw2025/storyforge
2. 点绿色的 **「Code」** 按钮 → 点 **「Download ZIP」**
3. 下载完是个压缩包，**右键 → 解压**到桌面（或你找得到的地方）
4. 解压后得到一个叫 **storyforge** 的文件夹，**双击进去**

### Windows 用户：

1. 在文件夹里找到 **`启动.bat`** 这个文件
2. **双击它**
3. 会弹出一个**黑色的窗口**，里面有中文提示，跟着它走就行：
   - 如果你电脑已经有运行环境 → 它会自动下载组件、启动、打开浏览器，**等一会儿就好**
   - 如果没有运行环境 → 它会提示帮你自动安装。**这时可能弹出一个"是否允许此应用更改你的设备"的窗口，点【是】**。装完它会让你**关掉窗口、再双击一次 `启动.bat`**，照做即可
4. 看到浏览器自动打开了 StoryForge，就成功了！
   （没自动打开就手动访问：`http://localhost:1111/storyforge/`）

> ⚠️ **如果双击没反应，或被杀毒软件（360/电脑管家/Defender）拦截、提示"风险"**：
> 这是因为杀软对这类自动脚本比较敏感，它是安全的。你可以在杀软弹窗里选「允许运行 / 信任」；
> 实在不行就用下面的**方式三**手动启动，或者干脆用**方式一网页版**。

### Mac 用户：

1. 在文件夹里找到 **`启动.command`** 这个文件
2. **右键点它 → 选「打开」**（第一次必须用右键打开，不能直接双击，否则系统会拦）
3. 弹出的提示里再点一次「打开」
4. 会出现一个**终端窗口**，跟着中文提示走（可能让你输入开机密码来装环境，**输密码时屏幕不显示字是正常的，输完按回车**）
5. 浏览器自动打开就成功了（没打开就手动访问 `http://localhost:1111/storyforge/`）

> 如果提示「权限不足 / permission denied」：打开「终端」App，输入 `chmod +x ` （后面留一个空格），把 `启动.command` 文件拖进终端窗口，按回车，再重新右键打开 `启动.command`。

### 以后再用：

- **不用再装环境**，直接双击 `启动.bat`（Windows）/ 右键打开 `启动.command`（Mac）即可
- 那个黑窗口/终端**不要关**，关了就停止运行；用完直接关掉它就停止

## ⌨️ 方式三：手动敲命令启动（最通用，一键启动失败时用这个）

如果一键启动被拦了或不好使，用这个最不会出问题的方法。两件事：**① 装 Node.js（一次）→ ② 在项目文件夹里敲两行命令**。

### 第 1 步：安装 Node.js（一台电脑只需装一次）

1. 浏览器打开 👉 https://nodejs.org/zh-cn
2. 点页面上的 **LTS** 版本下载（LTS = 长期支持版，更稳定）
3. 双击安装包，**一路点「下一步 / Next」直到完成**（Windows 如看到 "Add to PATH" 选项，保持勾选）
4. 装完**重启一下电脑**（保险，避免命令找不到）

### 第 2 步：在「项目文件夹里」打开命令窗口 ⭐（最关键，多数人卡这）

命令必须在 storyforge 文件夹"里面"敲，最不容易出错的方法：

**Windows：**
1. 双击进入 storyforge 文件夹（能看到 `package.json`、`src` 这些）
2. 看窗口**最上面那条显示路径的横条**（地址栏）
3. **鼠标左键点一下那条横条的空白处**，它会变成可编辑文字
4. 把里面内容删光，**打字输入 `cmd`，按回车**
5. ✅ 弹出的黑色窗口已经"站在"你的项目文件夹里了

**Mac：**
1. 打开「终端」App，输入 `cd ` （后面留一个空格）
2. 把 storyforge 文件夹**拖进终端窗口**，按回车
3. ✅ 终端已"站在"项目文件夹里了

### 第 3 步：装依赖（只需一次）

在命令窗口里输入下面这行，按回车，**等它跑完**（约 1~2 分钟，需联网）：

```
npm install
```

> 国内装不上/卡住，先输这行换国内源再 `npm install`：
> ```
> npm config set registry https://registry.npmmirror.com
> ```

### 第 4 步：启动

```
npm run dev
```

### 第 5 步：打开浏览器

访问 👉 **http://localhost:1111/storyforge/**

> 以后再用：进文件夹 → 地址栏输 `cmd` 回车（Mac 拖文件夹进终端）→ 输 `npm run dev` → 开浏览器。不用再装 Node 和依赖。

## ❓ 常见问题

**Q：我的小说会丢吗？**
A：数据存在浏览器本地，不上传服务器，**不清浏览器数据就一直在**。换电脑/浏览器不会同步，重要内容请用工具内「导出」备份。

**Q：提示 `npm 不是内部或外部命令` / `command not found`？**
A：Node.js 没装好。重装（Windows 勾选 Add to PATH），**装完重启电脑**再试。

**Q：怎么判断我是不是在对的文件夹里敲命令？**
A：Windows 看黑窗口那行路径结尾是不是 `\storyforge>`；不是就重做方式三第 2 步。

**Q：端口 1111 被占用？**
A：关掉占用的程序；或按命令窗口里实际显示的地址打开。

**Q：网页版和本地版有区别吗？**
A：功能一样。网页版免安装；本地版可离线。两者数据各自独立（各存各的浏览器）。
