📋 这个 Demo 展示的是什么？

这是 方案 4（AI → 结构化数据 → Canvas 渲染）的效果预览。

实际流程：用户在「设定库 → 世界观」中写下文字描述 → 点击「生成世界地图」→ AI 提取结构化 JSON → Canvas 渲染。

下方就是 AI 可能返回的结构化数据，直接渲染出的效果。所有地形、山脉、河流、城市标注都是代码绘制的（非图片）。
